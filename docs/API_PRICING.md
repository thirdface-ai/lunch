## Cost Optimization Tips

1. **Use Session Tokens for Autocomplete** - Bundles all keystrokes + Place Details into 1 session (FREE)
2. **Cache Place Details** - Avoid repeat calls for same places
3. **Cache Distance/Routes** - Avoid repeat calculations
4. **Use Text Search caching** - Same queries return cached results
5. **Request only needed fields** - Use field masks to reduce data tier costs
